== Description ==

The Travel Visa Widget plugin makes it easy to prepare for your trip by helping you determine which visa is needed for your destination. Simply enter your destination country and nationality to quickly get the necessary information about the paperwork required for your trip. With "Travel Visa Widget", you can rest assured that you have the necessary travel documents for your destination. Don't get caught up in the paperwork, use Travel Visa Widget to be prepared at all times and leave with peace of mind.

== Installation ==

Installing Travel Visa Widget is as easy as using it :

### INSTALL FROM WITHIN WORDPRESS

1. Visit the plugins page within your dashboard and select ‘Add New’
2. Search for ‘Travel Visa Widget’
3. Activate the Travel Visa Widget plugin through the ‘Plugins’ menu in WordPress

### INSTALL MANUALLY

1. Upload the ‘travel-visa-widget’ folder to the /wp-content/plugins/ directory
2. Activate the Travel Visa Widget plugin through the ‘Plugins’ menu in WordPress

### AFTER ACTIVATION

In the settings menu a new ‘travel Visa Widget’ submenu appears with some settings (Agency ID, Affilae ID).

== Frequently Asked Questions ==

= How do I change my settings? =

All options are located within Settings > Travel Visa Widget.

= Can I load the widget with a shortcode? =

Yes!
Simply put this shortcode wherever you want and.. that's it.

<code>
[travel_visa_widget]
</code>

= What to do if a country is not present or has no data? =

Our teams are automatically notified and the data will be updated accordingly.
